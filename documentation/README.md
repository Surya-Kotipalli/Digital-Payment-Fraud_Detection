# Documentation

This folder contains supporting project documentation, including the detailed technical report and presentation materials for the Digital Payment Fraud Detection project.

## Contents

- Project Report.pdf  
  Complete project documentation covering methodology, preprocessing, feature engineering, model training, evaluation metrics, and conclusions.

- DWDM Final PPT.pptx  
  Presentation slides summarizing the project objectives, workflow, models, results, and key insights.

These materials were developed as part of the Data Warehousing & Data Mining (DWDM) coursework.
